User Story: Display Login Form
Title: Display Login Form
Description: As a user, I want to see a login form with fields for email and password, so that I can enter my credentials to access the app.
Acceptance Criteria:
• The login form includes:
  – Email input field
  – Password input field
  – Login button
• Basic client-side validation:
  – Required fields
  – Proper email format
• Error messages are shown for invalid inputs
